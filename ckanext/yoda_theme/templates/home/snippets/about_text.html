YODA

El portal de datos en abierto YODA proporcina datos para el desarrollo de aplicaciones de ciudades inteligentes (e.g, datos de meteorología, ocupación de las ciudades). El portal de datos en abierto incluye fuentes de datos con información modelada siguiendo el estándar NGSI-LD (Next Generation Service Interface with Linked Data) y compatible con los FIWARE Smart Data Models. Los datoos publicados incluyen metadatos compatibles con el perfil Data Catalog Vocabulary Application Profile (DCAT-AP) v2.0.1. Además, sus datasets pueden ser harvesteados por otros portales a través del protocolo Open Archives Initiative Protocol for Metadata Harvesting (OAI-PMH) versión 2.
